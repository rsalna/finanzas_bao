# Ahorros 💳 (v2)

Incluye:
- Tope mensual de gastos libres
- Panel “Próximo pago BBVA” (corte y fecha límite configurables)
- Gráficas (Chart.js): por categoría, por método, últimos 6 meses

## Instalación
1) Copia la carpeta a `htdocs/finanzas_bao/`
2) Importa `sql/schema.sql` y `sql/seed.sql` (opcional).
3) Abre: http://localhost/finanzas_bao/index.php
